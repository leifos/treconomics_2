Practice Task

In order to get acquainted with the search system and newspaper database, we would like you to practice for a couple of minutes using the system to undertake a task similar to those you will need to undertake. Below is a description of the system.

The Search System and Search Task

The search system that you will use connects to a database of about 1.8 million newspaper articles from the New York Times. The articles within the database range from 1987 to 2007. The search system you'll be using looks similar to a web search engine.

For each search task, you'll be given a topic (like the one shown below in blue).

{% autoescape off %} {{ taskdescription }} {% endautoescape %}

Your goal is to find and save several relevant and different examples for the topic.

The system will let you enter queries and look at results, just like you would in a normal search engine.

When you view a document which contains a relevant example, then you can click the Bookmark button to save it.

If you need to remind yourself of the search task, you will be able to click the show task.

Once you find a number of relevant examples, you can click end task to proceed, otherwise the system will automatically time out after about 8 minutes.

After searching, you will be asked to list as many examples as possible that you have just found.

To get familiar with the system, we would like you to:

  • Select Show Task to remind you what the topic of the search task is.
  • Search for documents about this topic by entering queries.
  • Try entering a few different queries.
  • Save a document by bookmarking it.
  • Scroll down to click on the Next page button, and
  • Review your list of documents by selecting View Saved.
  • Note the system will automatically time out after 4 minutes and show you your performance.
